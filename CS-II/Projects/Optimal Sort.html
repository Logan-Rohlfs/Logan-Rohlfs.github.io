For this Project, I created a program that sorts an array of words in the most efficient way possible. I used my own algorithm very similar to the merge sort algorithm to sort the array. I then compared the efficiency of myalgorithm to the efficiency of the bubble sort algorithm. My sort algorithm is more efficient than the bubble sort algorithm because it has a time complexity of O(n log n) while the bubble sort algorithm has a time complexity of O(n^2). I also compared the efficiency of my algorithm to the efficiency of the quick sort algorithm. My algorithm is more efficient than the quick sort algorithm because it has a time complexity of O(n log n) while the quick sort algorithm has a time complexity of O(n^2). While it would have been easy to just copy the code off of the internet becuase merge sort is a very popular method of sorting data, all of the code (minus some help from W2Schools and StackOverflow for the basics) was written by me and my partners.
